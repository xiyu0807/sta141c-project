blbglm {blblm}R Documentation

Generalized linear regression with little bag of bootstraps

Description

Generalized linear regression with little bag of bootstraps

Usage

blbglm(formula, family, data, m = 10, B = 5000, parallel = FALSE)

Arguments

formula

a formula of model.

family

family for GLM

data

a data frame.

m

data splits, default = 10.

B

bootstraps, default = 5000.

parallel

logical value indicate use parallelization or not, default is not use.

Value

res A list including estimates and sigma.
